### 你的第一个基础应用
:::tip
先安装好需要的类库，再进行下一步, 不会的请看 [这里](/docs/getting-started/installing/)!
:::
以下示例，是Typescript所写。
<Steps>

1. 让我们来看一个可以尝试的最简单App，浏览器中的```Hello World```例子。
  
    你至少需要导入```StaticCanvas```和```FabricText```类。
    ```typescript
    import { StaticCanvas, FabricText } from 'fabric';
    ```

2. 然后实例化它们两个并将它们组合

    ```typescript
    const canvas = new StaticCanvas();
    const helloWorld = new FabricText('Hello world!');
    canvas.add(helloWorld);
    canvas.centerObject(helloWorld);
    ```

3. 然后你想用它来下载你的 PNG 文件

    ```typescript
    const imageSrc = canvas.toDataURL();

    // 下面是下载的代码
    const a = document.createElement('a');
    a.href = imageSrc;
    a.download = 'image.png';
    document.body.appendChild(a);
    a.click();
    document.body.removeChild(a);
    ```

4. 现在来试一试吧！

    复制下面代码或点击运行看看。
  
</Steps>
